# Comando: /coderabbit

# Ejecuta CodeRabbit CLI para una revisión profunda de los cambios.

Usage: `/coderabbit [--type uncommitted|committed|all]`

## Pasos

1. Ejecuta `coderabbit --type <tipo>` (por defecto 'all').
2. Reporta los hallazgos al usuario.

## Notas

- Optimizado para usuarios con suscripción Pro.
- Ideal para detectar bugs lógicos complejos antes del commit.
